  完整的计算属性:

     fullName:
     {
      set()
      {

      },
      get()
      {
           return this.firstName+this.lastName;
      }
     }

 计算属性的set方法一般不用，简化后如下
     fullName:
     {
      get()
      {
           return this.firstName+this.lastName;
      }
     }
 因为只有get方法，可以简化如下，从完整计算属性可以看出fullName本身是一个属性，是通过简化后成函数，所以用Mustache写法时，不用带括号:
     fullName()
     {
       return this.firstName+this.lastName;
     }
